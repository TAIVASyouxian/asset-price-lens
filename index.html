Install and deploy

Local testing
  1. Open a terminal in the asset-price-lens folder.
  2. Run python -m http.server 8000.
  3. Open http://localhost:8000.
GitHub Pages
  1. Create a GitHub repository and upload these files.
  2. Go to Settings, Pages.
  3. Select Deploy from a branch, choose the main branch and root folder, then save.
Vercel
  1. Import the repository in Vercel.
  2. Use the static default settings. No build command is required.
  3. Deploy and open the generated URL.
Phone install
  1. iPhone Safari: Share, Add to Home Screen, Add.
  2. Android Chrome: Menu, Add to Home screen or Install app.
Update later
  1. Edit the files and increase the app version in app.js, index.html, and service-worker.js.
  2. Deploy the changed files.
  3. Open the app and tap Reload app / Clear cache.
