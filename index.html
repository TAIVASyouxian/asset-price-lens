安裝與部署

本機測試
  1. 在 asset-price-lens 資料夾開啟終端機。
  2. 執行 python -m http.server 8000。
  3. 開啟 http://localhost:8000。
GitHub Pages
  1. 建立 GitHub repository，並上傳這些檔案。
  2. 進入 Settings，Pages。
  3. 選擇從 branch 部署，選 main branch 與 root folder，然後儲存。
Vercel
  1. 在 Vercel 匯入 repository。
  2. 使用靜態網站預設設定，不需要 build command。
  3. 部署後開啟產生的網址。
手機安裝
  1. iPhone Safari：分享，加入主畫面，新增。
  2. Android Chrome：選單，加入主畫面或安裝應用程式。
日後更新
  1. 修改檔案，並同步提高 app.js、index.html 與 service-worker.js 的版本。
  2. 部署更新後的檔案。
  3. 開啟 app，點選重新載入 / 清除快取。

如果手機版沒有更新，請先按此按鈕重新整理快取。若仍未更新，請用 Safari 重新開啟網址，或刪除主畫面圖示後重新加入。

本工具僅供個人估算使用。匯率與 ETF 價格僅供約略換算，不構成投資建議、交易訊號、金融建議或保證市場資料。做出金融或消費決策前，請自行確認實際價格。
